# SaaS Product Analytics Portfolio
## AI-Enhanced Business Intelligence for Software Products

### Project Overview
Comprehensive analytics platform demonstrating product intelligence capabilities for SaaS genomics software. Built to showcase modern BI stack integration with AI/ML capabilities.

### Objective
Analyze user behavior, subscription health, and product usage patterns to drive data-driven product decisions.

### Tech Stack
- **Data Warehouse**: PostgreSQL / DuckDB
- **Transformation**: dbt
- **Visualization**: Tableau
- **ML/AI**: Python (scikit-learn, Prophet), OpenAI API
- **Demo App**: Streamlit
